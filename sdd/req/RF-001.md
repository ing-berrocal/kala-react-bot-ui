### RF-001: Dashboard de Comentarios (Index)

**Prioridad:** Alta

#### API Contract
```javascript
// GET /api/comments?date={YYYY-MM-DD}&status={STATUS}
// Response:
[
    {
        "comment": {
            "from": {
                "id": string,
                "name": string
            },
            "message": string,
            "created_time": number
        },
        "response": {
            "message": string
        },
        "status": string,
        "comment_id": string,
        "post_id": string,
        "parent_id": string
    }
]
```

#### Componentes a Implementar
- `CommentsDashboard.jsx` - Contenedor principal
- `CommentsFilter.jsx` - Panel de filtros
- `CommentsTable.jsx` - Tabla de comentarios
- `CommentRow.jsx` - Fila individual

#### Componentes RSuite
- `<DatePicker>` para selección de fecha
- `<SelectPicker>` para filtro de estado
- `<Button>` para aplicar filtros

#### Lógica de Negocio Frontend
- Fecha por defecto: fecha actual del navegador
- Estados por defecto: todos (PENDING, WAITING, PROCESSED)
- Actualización automática cada 30 segundos (opcional)
- Paginación: 20 comentarios por página


#### Estado del Componente
```javascript
const [filters, setFilters] = useState({
  date: new Date(),
  status: 'ALL'
});
```

#### Validaciones
- La fecha no puede ser futura
- El estado debe ser uno de: ALL, PENDING, WAITING, PROCESSED
