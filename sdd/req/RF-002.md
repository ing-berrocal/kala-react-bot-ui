### RF-003: Gestión de Publicaciones

**Prioridad:** Alta

#### API Contracts
```javascript
// GET /api/posts
// Response:
{
  data: [
    {
      post_id: "post_456",
      plataforma: "Facebook",
      fecha_creacion: "2026-04-20T14:00:00Z",
      producto: "Cuenta de Ahorro",
      estado: "ACTIVE"
    }
  ]
}

// PUT /api/posts/{post_id}
// Request:
{
  plataforma: "Instagram",
  producto: "Tarjeta de Crédito"
}

// PATCH /api/posts/{post_id}/deactivate
// Response:
{
  post_id: "post_456",
  estado: "INACTIVE"
}
```

#### Componentes a Implementar
- `PostsManager.jsx` - Contenedor principal
- `PostsTable.jsx` - Tabla de publicaciones
- `PostRow.jsx` - Fila con acciones
- `EditPostModal.jsx` - Modal de edición
- `DeactivateConfirmation.jsx` - Diálogo de confirmación

#### Validaciones de Formulario
- Plataforma: requerida, solo "Facebook" o "Instagram"
- Producto: requerido, texto no vacío
