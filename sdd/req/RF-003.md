### RF-003: Interfaz de Chat

**Prioridad:** Alta

#### API Contracts
```javascript
// POST /api/chat/messages
// Request:
{
  message: "Hola, necesito ayuda",
  sessionId: "session_789" // opcional en primer mensaje
}

// Response:
{
  reply: "Hola, ¿en qué puedo ayudarte?",
  sessionId: "session_789",
  timestamp: "2026-04-22T11:00:00Z"
}

// GET /api/chat/history?sessionId={sessionId}
// Response:
{
  messages: [
    {
      id: "msg_1",
      sender: "user",
      content: "Hola",
      timestamp: "2026-04-22T10:59:00Z"
    },
    {
      id: "msg_2",
      sender: "agent",
      content: "Hola, ¿en qué puedo ayudarte?",
      timestamp: "2026-04-22T11:00:00Z"
    }
  ]
}
```

#### Componentes a Implementar
- `ChatInterface.jsx` - Contenedor principal
- `MessageList.jsx` - Lista de mensajes
- `Message.jsx` - Mensaje individual (user/agent)
- `MessageInput.jsx` - Input de texto + botón enviar
- `TypingIndicator.jsx` - Indicador "escribiendo..."

#### Estado del Chat
```javascript
const [state, setState] = useState({
  messages: [],
  sessionId: null,
  currentMessage: '',
  sending: false,
  error: null
});
```

#### Comportamiento UX
- Auto-scroll al último mensaje
- Enter envía mensaje (Shift+Enter = nueva línea)
- Deshabilitar input mientras se envía
- Mostrar "escribiendo..." durante respuesta
- Timestamp en cada mensaje
